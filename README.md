# JKagiDesigns LLC Website

Professional business website built with pure HTML, CSS, and JavaScript (no frameworks).

## Features

### Pages
- **Home Page** (`index.html`) - Animated hero section with gradient background, about section, features, and statistics
- **Services Page** (`services.html`) - Service offerings and three-tier pricing structure
- **Contact Page** (`contact.html`) - Contact form with validation and business information
- **RSS Feed Page** (`rss.html`) - IT and programming news from multiple sources with filtering

### Technical Highlights
- Pure HTML5, CSS3, and vanilla JavaScript
- Fully responsive design (mobile, tablet, desktop)
- Smooth animations and transitions
- Interactive navigation with mobile hamburger menu
- Form validation
- RSS feed integration with fallback mock data
- Modern gradient color scheme
- Card-based layouts
- Intersection Observer API for scroll animations

## Getting Started

Simply open `index.html` in a web browser, or serve the files with any web server:

```bash
# Using Python
python3 -m http.server 8080

# Using Node.js
npx http-server

# Then open http://localhost:8080
```

## File Structure

- `index.html` - Home page
- `services.html` - Services and pricing page
- `contact.html` - Contact page with form
- `rss.html` - RSS news feed page
- `styles.css` - Global styles and responsive design
- `script.js` - Main JavaScript functionality
- `rss.js` - RSS feed handler
